# CS Fundamentals

Добро пожаловать в руководство по фундаментальным основам Computer Science.

## Что вы найдете здесь

- **Алгоритмы** — основы алгоритмов, сложность, сортировки, поиск
- **Структуры данных** — массивы, списки, деревья, графы, хэш-таблицы
- **Архитектура ЭВМ** — устройство компьютера, память, процессор, ввод-вывод
- **Дискретная математика** — логика, множества, комбинаторика, теория графов

## Быстрый старт

Эти знания остаются актуальными десятилетиями и являются основой для понимания любых современных технологий.

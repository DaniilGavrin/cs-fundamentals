# Структуры данных

## Основные структуры

### Массивы (Arrays)
- Последовательное хранение элементов
- Быстрый доступ по индексу O(1)
- Медленная вставка/удаление в середине

### Связные списки (Linked Lists)
- Элементы связаны указателями
- Быстрая вставка/удаление
- Медленный доступ по индексу O(n)

### Стеки (Stacks)
- LIFO (Last In, First Out)
- Операции: push, pop, peek

### Очереди (Queues)
- FIFO (First In, First Out)
- Операции: enqueue, dequeue

### Деревья (Trees)
- Иерархическая структура
- Бинарные деревья поиска (BST)
- Сбалансированные деревья (AVL, Red-Black)

### Графы (Graphs)
- Вершины и рёбра
- Ориентированные и неориентированные
- Алгоритмы обхода: DFS, BFS

### Хэш-таблицы (Hash Tables)
- Быстрый поиск, вставка, удаление O(1)
- Использование хэш-функций

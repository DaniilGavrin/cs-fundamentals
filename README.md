# Computer Science Fundamentals

Полное руководство по фундаментальным основам компьютерных наук. Здесь собраны вечные концепции, которые не устаревают со временем и являются базой для любого разработчика.

## 📚 Содержание

- **Алгоритмы и структуры данных** — основы эффективности кода
- **Дискретная математика** — логика, множества, графы, комбинаторика
- **Архитектура компьютера** — как работает железо
- **Сетевая модель OSI и TCP/IP** — фундамент сетевого взаимодействия
- **Операционные системы** — процессы, память, файловые системы
- **Теория вычислений** — автоматы, машины Тьюринга, сложность
- **Кодировки и представление данных** — от битов до Unicode
- **Регулярные выражения** — универсальный инструмент поиска паттернов

## 🚀 Быстрый старт

### Локальный запуск документации

```bash
# Установка зависимостей
pip install mkdocs mkdocs-material

# Запуск локального сервера
mkdocs serve
```

Документация будет доступна по адресу: http://127.0.0.1:8000

### Генерация PDF шпаргалок

В каждом разделе есть `.md` файлы, которые можно сконвертировать в PDF с помощью нашего универсального скрипта:

```bash
python generate_git_pdf.py -i docs/algorithms.md -o algorithms.pdf
```
